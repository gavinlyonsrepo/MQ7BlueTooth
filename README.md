# MQ7BlueTooth

Overview
--------------------
* Name : MQ7BlueTooth

* Description : 

A project to read an MQ7 carbon monoxide(CO) sensor and transmits its data via bluetooth module.

* Author: Gavin Lyons

Software
------------------------------

*Libraries*

* [MQ7.h](https://github.com/fjebaker/MQ7)
* <SoftwareSerial.h> // Arduino built-in for the bluetooth comms
